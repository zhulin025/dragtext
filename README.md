# Drag Text String

这是一个纯前端静态页面，用一篇“当前大模型与 AGI 的进程”文章实现类似文字绳子的拖拽效果。

入口文件：

- `index.html`

## 实现方式

这个网页主要使用 **HTML + CSS + 原生 JavaScript** 实现，没有引入第三方开源库。

核心思路如下：

1. **把文章拆成单个字**

   页面用 `Intl.Segmenter` 把中文文章按 grapheme 拆开，每个字生成一个独立的 `span.letter`，然后用 `position: absolute` 放到页面上。

2. **手动排版**

   脚本用 `canvas.measureText()` 测量每个字的宽度，再按容器宽度自动换行。为了让文字像一根连续绳子，代码把每一行按“蛇形顺序”连接：一行从左到右，下一行从右到左。

3. **物理模拟**

   每个字都被当成一个小质点，保存当前位置 `x/y` 和上一帧位置 `px/py`。动画循环里用的是简化版 **Verlet 积分**：

   - 根据上一帧位移推算速度
   - 加上重力
   - 多次迭代修正相邻字之间的距离
   - 让字和字之间保持像绳子一样的长度约束

4. **拖拽交互**

   末尾几个字初始是可拖动的。用户拖动尾部时，拉力超过阈值后，前一个字会被“解锁”，然后继续往前传递，于是整篇文章会像被从排版里抽出来一样。

5. **视觉层**

   CSS 负责纸张感背景、字体、按钮和末尾提示圈。页面还用了一个 `canvas` 画很淡的连接线，让文字链更像一根绳子。

## 主要技术

- HTML
- CSS
- 原生 JavaScript
- DOM 逐字渲染
- Canvas 文本测量与连接线绘制
- Pointer / Mouse / Touch 事件
- Verlet 物理积分
- 距离约束模拟

## 开源依赖

当前版本没有使用第三方开源项目或 npm 依赖。参考网页本身也是类似的原生 DOM + Verlet 物理约束思路；本项目把逻辑写在单个 `index.html` 文件中。
